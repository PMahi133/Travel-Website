# Travel-Website
A full-stack travel website built using PHP and MySQL where users can explore seasonal travel packages, book trips, and manage their profiles. Includes an admin panel to manage packages, bookings, and user feedback.

# Database name: passion1
You can change the database name in config file as per your setup.

## Features
###  User Side
- Home page with travel overview
- About page
- Seasonal packages:
  - Winter packages
  - Summer packages
  - Monsoon packages
- Package details page
- Booking and payment system
- User dashboard:
  - View bookings
  - Profile management
- Gallery (past customer experiences)
- Contact form
- Privacy policy
- Login & Signup system

### Admin Side
- Admin dashboard
- View user bookings
- Add new places and travel packages
- View customer feedback

## Technologies Used
- PHP
- MySQL
- HTML
- CSS
- JavaScript
- XAMPP (for local server)

##  How to Run the Project

1. Install XAMPP
2. Place the project folder inside: htdocs Folder 
3. Start Apache and MySQL
4. Import the database file (`passion1.sql`) using phpMyAdmin
5. Open browser and go to: http://localhost/your-project-folder
